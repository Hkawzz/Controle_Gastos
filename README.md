# Controle de Gastos

Projeto em Django, que permite ter uma visão geral e simples o estado financeiro pessoal.

## Origem do projeto
Eu usava uma planilha financeira pessoal, que eu mesmo criei, agora que venho aprendendo Django, pensei em transformar ela em um Site Web.

## Funcionalidades
- Registrar ganhos e gastos.
- Cálculo automático do dinheiro disponivel.
- Organização por categorias
- Persistência em banco de dados

## Arquivo de referência
Deixei o arquivo da planilha original, que usei como base em '/planilha'
